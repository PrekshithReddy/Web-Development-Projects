# 🔐 ExpressJS Secrets

A simple password-protected web application built with **Node.js** and **Express.js**.

This project demonstrates the fundamentals of Express by handling form submissions and conditionally serving a protected HTML page based on user input.

> **Course Project:** Built as part of *Dr. Angela Yu's Complete Web Development Bootcamp*.

---

## 🚀 Features

- Express.js web server
- Handles HTTP POST requests
- Password-based access control
- Serves static HTML pages
- Simple and beginner-friendly project structure

---

## 🛠️ Technologies Used

- Node.js
- Express.js
- HTML5

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/PrekshithReddy/Web-Development-Projects.git
```

Navigate to the project folder:

```bash
cd Web-Development-Projects/project-9-ExpressJs-Secrets
```

Install dependencies:

```bash
npm install
```

Start the application:

```bash
node index.js
```

Open your browser and visit:

```
http://localhost:3000
```
